# 🎵 Audio Equalizer 

Веб-приложение на FastAPI для захвата системного звука или звука отдельного окна/вкладки с визуализацией в виде эквалайзера в реальном времени.

![Python](https://img.shields.io/badge/Python-3.10+-blue)
![FastAPI](https://img.shields.io/badge/FastAPI-0.115-green)
![License](https://img.shields.io/badge/License-MIT-yellow)

## ✨ Возможности

- 🔊 **Захват всего системного звука** (через WASAPI loopback)
- 🪟 **Захват звука конкретного окна или вкладки** (через браузерный Screen Capture API)
- 📊 **Визуализация эквалайзера** в реальном времени с минимальной задержкой
- 🎨 **Гибкая настройка внешнего вида:**
  - Цвета столбиков (градиент верх/низ)
  - Радужный режим (цвет по частоте)
  - Цвет фона
  - Прозрачный фон (для использования в OBS)
  - Регулировка прозрачности
- 🎚 **Настройка чувствительности** — чтобы эквалайзер не зашкаливал
- 🔗 **Эквалайзер доступен по отдельной ссылке** — удобно для трансляций и второго монитора


## 📖 Использование

1. Откройте главную страницу: `http://127.0.0.1:8000/`
2. Выберите источник звука:
   - **🔊 Весь компьютер:** в выпадающем списке выберите loopback-устройство (обычно это ваши динамики или наушники).
   - **🪟 Окно / вкладка приложения:** для захвата звука конкретного окна или вкладки браузера.
3. Нажмите кнопку **▶ Начать захват**.
4. Откройте страницу визуализации, нажав на ссылку **📊 Открыть эквалайзер →** (или перейдите по адресу `http://127.0.0.1:8000/equalizer` в отдельной вкладке или окне).

### ⚠️ Особенности захвата окна/вкладки

Если вы выбрали режим захвата конкретного окна, обратите внимание на следующие важные моменты:

- ✅ В появившемся системном диалоге **обязательно включите галочку «Поделиться аудио»** (Share audio), иначе звук не будет захвачен.
- 🚫 **Не закрывайте вкладку захвата** — именно она обрабатывает звук и передаёт данные эквалайзеру.
- 🌐 Страницу эквалайзера (`http://127.0.0.1:8000/equalizer`) можно свободно открывать в других вкладках или окнах.

> 💡 **Совет для стримеров:** Эквалайзер можно добавить в **OBS Studio** как *Browser Source* (Браузер). Включите в настройках эквалайзера **«Прозрачный фон»** — и вы получите красивый анимированный оверлей для ваших стримов!
